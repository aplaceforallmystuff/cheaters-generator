# CLAUDE.md - cheaters-generator

Generate personal Claude Code cheatsheet reference files.

## Overview
Tool for creating quick-reference cheatsheets for Claude Code commands, skills, plugins, and MCP servers.

## Usage
Generates markdown cheatsheet files for personal reference.

## Constraints
```yaml
rules:
  - id: personal-use
    description: Output is for personal reference, not documentation
  - id: concise
    description: Keep entries brief - command/purpose only
```
