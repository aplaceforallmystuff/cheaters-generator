About Info

Claude Code Quick-Reference — Your personalized command reference

What Is This?

Quick Reference System
A local HTML-based reference for your Claude Code setup — skills, agents, commands, and MCP servers all in one searchable place.
Inspired by Brett Terpstra's Cheaters
Auto-Discovery
Scans your actual ~/.claude/ configuration to build sheets tailored to YOUR setup, not generic documentation.
Skills, agents, commands, MCP servers, plugins

Keyboard Shortcuts

KeyAction
/Focus search
EscClose search
↑ or kPrevious sheet
↓ or jNext sheet
EnterSelect search result

How to Use

/generate-quickref
Generate a new quick-reference from scratch. Scans your configuration and creates all sheets.
/generate-quickref [optional: output directory]
/sync-quickref
Update an existing quick-reference with changes. Shows diff of added/removed/changed items.
/sync-quickref [optional: quickref directory]

Running Locally

Option 1: Open index.html directly in your browser

Option 2: Serve locally for development:
cd ~/Dev/claude-code-quickref && python3 -m http.server 8888
Then visit http://localhost:8888

Rebuilding After Edits

If you manually edit sheet files in sheets/, rebuild the main.js:

node scripts/build.js
